skxray.constants.PowderStandard¶

class skxray.constants.PowderStandard(name, reflections)[source]¶

Class for providing safe access to powder calibration standards data.

Parameters:

name : str

Name of the standard

reflections : list

A list of (d, (h, k, l), q) values.

Attributes

name Name of the calibration standard
reflections List of the known reflections

Methods

from_d(name, d[, hkl]) Method to construct a PowderStandard object from known \(d\) values.
