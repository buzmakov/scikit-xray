skxray.core.MD_dict¶

class skxray.core.MD_dict(md_dict=None)[source]¶

A class to make dealing with the meta-data scheme for DataExchange easier

Examples

Getting and setting data by path is possible

>>> tt = MD_dict()
>>> tt['name'] = 'test'
>>> tt['nested.a'] = 2
>>> tt['nested.b'] = (5, 'm')
>>> tt['nested.a'].value
2
>>> tt['nested.a'].units is None
True
>>> tt['name'].value
'test'
>>> tt['nested.b'].units
'm'
